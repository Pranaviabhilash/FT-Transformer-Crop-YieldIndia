# FT-Transformer Crop Yield Prediction (India)

This repository contains implementation of FT-Transformer for crop yield prediction using multi-source agricultural data across Indian states.

## Dataset

Crop Yield Dataset:
https://www.kaggle.com/datasets/anshumish/crop-yield-data-with-soil-and-weather-dataset

## Features Used

* Soil nutrients (N, P, K, pH)
* Weather variables (Rainfall, Temperature, Humidity)
* Crop metadata (State, Season, Crop type)

## Models Compared

* Random Forest
* XGBoost
* TabTransformer
* FT-Transformer (Proposed Model)

## Results

FT-Transformer achieved best performance:

* MAE: 0.8914
* RMSE: 3.0112
* R²: 0.9263
